# 坦克大战 (Tank Battle)

一个使用纯 HTML、CSS 和 JavaScript 制作的坦克大战小游戏。

## 在线试玩

直接用浏览器打开 `index.html` 即可开始游戏。

## 操作说明

- **WASD** 或 **方向键**：移动坦克
- **空格键**：发射炮弹
- 消灭所有敌方坦克进入下一关
- 被敌方炮弹击中会损失生命，生命耗尽则游戏结束

## 游戏特性

- 玩家坦克与敌方 AI 坦克
- 随机生成的墙体地图
- 子弹碰撞检测
- 得分、生命、关卡系统
- 游戏结束与重新开始

## 本地运行

```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
cd tank-battle
open index.html
```

或使用任意本地静态服务器：

```bash
npx serve .
```
